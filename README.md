# EuDorama

Frontend React + Vite focado em:
- controle de clientes e assinaturas
- gestão financeira e renovações
- integração com Supabase

## Requisitos

- Node.js 18+

## Desenvolvimento

1. Instale as dependências:
   `npm install`
2. Configure variáveis em `.env.local`:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Rode localmente:
   `npm run dev`

## Build

- `npm run build`
- `npm run preview`
